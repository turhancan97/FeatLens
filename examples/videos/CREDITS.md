# Example video credits

`cockatoo.mp4` is the standard sample clip distributed with the
[imageio](https://imageio.readthedocs.io/) project
([imageio-binaries](https://github.com/imageio/imageio-binaries/blob/master/images/cockatoo.mp4),
fetchable as `imageio.imread("imageio:cockatoo.mp4")`). The copy here is trimmed to a short
segment and downscaled to 480×272 to keep the repository small; it is used only to demonstrate the
temporal V-JEPA path in [`examples/vjepa_video.py`](../vjepa_video.py).
